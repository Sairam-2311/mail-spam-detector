# Email Spam Detector
A simple machine learning web application that classifies emails as **Spam** or **Not Spam**.
# Built With
* Python
* Flask
* Scikit-learn
* HTML
* CSS
# How to Run
```bash
pip install -r requirements.txt
python app.py
```
Open:
```text
http://127.0.0.1:5000
```
